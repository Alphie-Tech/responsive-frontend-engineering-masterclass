## 🧩 DAILY PRACTICE BREAKDOWN

### 🗓 Monday — HTML Foundations

**Folder:** `📁 monday/`

#### 🛠 Exercise 1 — HTML Boilerplate

* **File:** `index.html`
* **Requirements:**
* `<!DOCTYPE html>`
* `<html lang="en">`
* `<head>` and `<body>`
* **Meta tags:** `charset`, `viewport`, `description`, `author`
* `<title>`



#### 🛠 Exercise 2 — Content Structure

* **Inside `<body>` include:**
* Headings (`h1`–`h3`)
* Paragraphs
* Ordered list
* Unordered list
* Links
* Images
* Comments


* **Required Sections:**
* Hero Introduction
* About Me
* Learning Goals
* Favorite Technologies
